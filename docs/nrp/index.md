# National Research Platform

The [National Research Platform](https://nrp.ai/){:target="_blank"} {{ ext }} is a community-owned research and education platform built on a global Kubernetes cluster. The NRP provides access to cutting-edge technologies in AI, HPC, data storage and networking. LBNL scientists and researchers are eligible to make use of the National Research Platform.

!!! note

    This page provides LBNL-specific guidance for accessing NRP. For full NRP platform documentation, please refer to the official [NRP documentation](https://nrp.ai/documentation/){:target="_blank"} {{ ext }}.

## Getting Started

Please follow the instructions on the [Getting Started](https://nrp.ai/documentation/userdocs/start/getting-started/){:target="_blank"} {{ ext }} documentation page of NRP to create an account on NRP Nautilus by logging in to the NRP Nautilus portal. As stated in the NRP Getting Started page, eligible faculty members and researchers can request to become namespace admin for their research group. At LBNL, you can do so by emailing ScienceIT.

## Becoming a Namespace Admin

If you are a scientist or researcher at LBNL who needs your own namespace (for your group), please send an email to [scienceithelp@lbl.gov](mailto:scienceithelp@lbl.gov) with a request to be promoted to a namespace admin. In the request, please include:

* Your name, division and department
* The desired name of your group namespace
* A brief description of your intended usage

Once you become a namespace admin, you can create your own namespaces and add other users using the [Namespaces manager](https://nrp.ai/namespaces){:target="_blank"} {{ ext }}.

## Managed Services Available on NRP

NRP has a number of services that are already deployed by cluster admins. These are especially useful if you do not want to run your own containers, but want to run computations on a Jupyter notebook or interact with hosted open-weight Large Language Models (LLMs). Some popular deployed services are:

* [JupyterHub on Nautilus](https://nrp.ai/documentation/userdocs/jupyter/jupyterhub-service/){:target="_blank"} {{ ext }}
* [NRP-Managed LLMs](https://nrp.ai/documentation/userdocs/ai/llm-managed/){:target="_blank"} {{ ext }}
* [Coder on Nautilus](https://nrp.ai/documentation/userdocs/coder/coder/){:target="_blank"} {{ ext }}
